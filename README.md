# Quark

**One search bar for everything.**

Quark is a universal search engine that lets you search across Gmail, Google Drive, local files, bookmarks, GitHub, Canva, Notion, Slack, and more—all from a single, lightning-fast interface.

## Features

- ⚡ **Lightning Fast** - Search results in under 0.4s
- 🔒 **Privacy First** - End-to-end encrypted, your data stays yours
- 🤖 **AI-Powered** - Smart search that understands context
- 🔌 **Multiple Integrations** - Gmail, Drive, GitHub, Notion, Slack, Canva, and more
- 💻 **Local + Cloud** - Search your entire computer and cloud apps simultaneously
- 🌓 **Dark Mode** - Easy on the eyes, day or night

## Quick Start
```bash
# Clone the repository
git clone https://github.com/junaidmirani/quark.git

# Install dependencies
cd quark
npm install

# Start the app
npm start
```
